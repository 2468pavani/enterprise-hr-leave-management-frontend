# Enterprise HR Leave Management Portal – Frontend System

This project is a real-time enterprise HR Management System (HRMS) frontend application developed using HTML5 and CSS3.  
It simulates how employee portals work in real corporate environments.

---

## Project Overview

In real organizations, HR systems are part of a larger employee portal.  
This project represents the Leave Management module integrated inside an Employee Portal.

After login, employees can access:
- Dashboard
- Attendance
- Leave Management
- Payroll

## Module Description

### Login Page
- Entry point to the employee portal
- Enterprise authentication UI
- Responsive layout using HTML and CSS

### Dashboard
- Central employee portal
- Navigation to all HR modules
- Sidebar-based enterprise layout

### Attendance Module
- Displays employee attendance details
- Monthly attendance overview

### Leave Management Module
- Apply leave
- View leave balance
- Track leave requests

### Payroll Module
- Displays salary details
- Monthly payroll breakdown
